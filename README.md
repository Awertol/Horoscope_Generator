# 🌌 Astral Guide

Moderní desktopová aplikace v **WPF (C#)** pro zjištění svého osudu, nabízející UI s mystickou estetikou.

<img width="439" height="568" alt="horoscope_preview" src="https://github.com/user-attachments/assets/10536ff5-f953-4d83-ae6e-9f47c17663ca" />

### ✨ Vlastnosti
* **Moderní UI** – gradienty, průhlednost a zaoblené rohy
* **Přesné horoskopy** – určitě se v nich najdete i Vy!
* **Napojení na vesmír** – naše tajná přísada
* **Responzivita** – adaptivní layout, aplikaci můžete roztáhnout, zmenšit, zvětšit

### 🛠 Technologie
* **C# / .NET / WPF**
* **XAML** (Vlastní stylování)
* **JetBrains Rider** (IDE)

### 🚀 Spuštění
Hotový soubor k instalaci naleznete v sekci **[Releases](../../releases)**. Stačí stáhnout, rozbalit a spustit `.exe`.
